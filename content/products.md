## Cuesheet

**An app for indoor-cycling and spin instructors. Design a class, run it
hands-free.**

Teaching a great indoor-cycling class takes planning. An instructor maps out
the ride — warm-up, climbs, sprints, recovery — and times each block to the
music. Most do this on paper, in a notes app, or from memory, then spend the
class watching the clock instead of the room.

Cuesheet replaces that. You build the ride in advance: lay out the segments,
set the effort and cadence for each, and line them up with your playlist. When
it is time to teach, Cuesheet runs the plan for you — cueing each block in turn
so you can focus on the class rather than the timing.

### What it does

- **Plan a class in advance.** Build a structured ride from warm-up to
  cool-down, with the effort, cadence, and duration for every segment.
- **Run it hands-free.** Cuesheet steps through the plan and cues each block,
  so you teach instead of tracking the clock.
- **Match the music.** Line segments up with your playlist so the work follows
  the beat.
- **Reuse and adapt.** Save the rides you like and adjust them for the next
  class instead of starting from scratch.

Cuesheet is built for iPhone and iPad.

Learn more at [cuesheet.ca](https://cuesheet.ca).
